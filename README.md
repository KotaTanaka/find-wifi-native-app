## Technology

* Expo
* React Native
* TypeScript

## Requirement

* Node `v12`
* Yarn

## Getting Started

#### Install

```bash
$ git clone git@github.com:KotaTanaka/find-wifi-native-app.git
$ cd find-wifi-native-app
$ yarn
```

#### Run

```bash
$ yarn start
```
